# 🍀 로또 번호 생성기

원하는 개수만큼 100% 중복되지 않는 로또 번호 조합을 생성하는 간단한 웹 애플리케이션입니다.

🔗 **[지금 바로 사용해보기 (Live Demo)](https://jeju-mandarin.github.io/lotto)**
*(위 링크의 `lotto` 부분은 실제 생성하신 저장소 이름으로 변경해주세요.)*

---

## ✨ 주요 기능

이 생성기는 JavaScript의 `Set` 자료구조를 활용하여 두 가지 유형의 중복을 모두 방지합니다.

* **원하는 개수만큼 생성:** 1개부터 100개까지 원하는 수량의 로또 번호 조합을 한 번에 생성할 수 있습니다.
* **완벽한 중복 제거 (핵심 기능):**
    1.  **한 줄 내 중복 제거:** `Set`을 사용해 한 줄(세트)에 6개의 숫자가 절대 겹치지 않습니다.
    2.  **줄 간의 중복 제거:** `Set`을 한 번 더 사용하여, 생성된 여러 줄의 번호 조합(예: `1, 2, 3, 4, 5, 6`)이 서로 겹치지 않도록 보장합니다.
* **자동 정렬:** 생성된 모든 번호 조합은 보기 편하도록 오름차순으로 자동 정렬되어 표시됩니다.
